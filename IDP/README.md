# Victoria Identity Provider – API

## Overview
This project is a learning-oriented Identity Provider API built in .NET.  
It focuses on understanding authentication flows and gradually building the core components of an IDP.

## Current Features
- **Basic user login**  
  Simple username/password authentication flow.

- **Refresh token system**  
  Full refresh token rotation and validation.

- **User-Agent capture**  
  Early implementation, still in progress.

- **Initial API structure**  
  Foundation for future authentication and security features.

## Work in Progress
- Improving validation and error handling  
- Expanding User-Agent and client context extraction  
- Adding RSA signing for access tokens  
- Adding email confirmation and password reset flows  
- General cleanup and refinement as learning progresses

## Purpose
This project is created for learning and experimenting with authentication concepts in .NET.  
It evolves step by step as new techniques and security mechanisms are explored.
